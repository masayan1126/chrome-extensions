## Detailed Description

Copy Tab Anything is a powerful Chrome extension for exporting browser tabs as formatted lists. Perfect for researchers, developers, content creators, and anyone managing multiple tabs.

### Key Features

**Multiple Export Formats**
- Markdown: `[title](url)`
- HTML: `<a href="url">title</a>`
- Plain text: `title - url`
- Title + URL: Line-separated format
- Custom templates: Create your own format!

**Flexible Scope Options**
- Current window tabs
- All windows tabs
- Specific tab groups (unique feature!)
- Custom selection

**Advanced Options**
- URL decoding for international characters
- Punycode decoding for internationalized domain names
- Tab group support

**Quick Access**
- Extension popup
- Right-click context menu
- Keyboard shortcut: `Alt+Shift+C`

### Use Cases

- **Researchers**: Save reference lists for papers
- **Developers**: Share debugging tabs with team members
- **Content Creators**: Build link collections for articles
- **Students**: Organize study materials
- **Project Managers**: Export task-related resources

### Privacy

- No data collection
- Everything runs locally in your browser
- No external servers
- Open source
