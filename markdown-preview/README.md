# Rich Markdown Preview

Markdownファイルをリアルタイムでプレビューするツール

## 概要

このプロジェクトは、Markdownファイルを編集しながらリアルタイムでプレビューを表示するツールです。

## 機能

- Markdownのリアルタイムプレビュー
- シンタックスハイライト対応
- レスポンシブデザイン

## セットアップ

```bash
# リポジトリのクローン
git clone https://github.com/masayan/markdown-preview.git
cd markdown-preview

# 依存関係のインストール
npm install

# 開発サーバーの起動
npm start
```

## 使い方

1. アプリケーションを起動
2. Markdownファイルを開く
3. 編集内容がリアルタイムでプレビューに反映されます
