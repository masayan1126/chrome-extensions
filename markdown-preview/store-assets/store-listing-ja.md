# Chrome Web Store 掲載情報（日本語）

## 拡張機能名
Markdown Preview

## 短い説明（132文字以内）
ローカルのMarkdownファイルを美しくプレビュー。カスタマイズ可能なテーマ、フォント選択、目次の自動生成に対応。

## 詳細な説明

Markdown Previewは、ローカルのMarkdownファイルを美しく読みやすくプレビューするためのChrome拡張機能です。

### 主な機能

📖 **美しいプレビュー**
- Markdownファイルを見やすいフォーマットで表示
- シンタックスハイライト付きのコードブロック
- テーブル、リスト、引用、画像の美しいレンダリング

🎨 **カスタマイズ可能なテーマ**
- ダーク/ライトモードのプリセットテーマ
- 完全にカスタマイズ可能なカラースキーム
- 見出し、リンク、コードブロックなど要素ごとの色設定

🔤 **フォント選択**
- 6種類のフォントファミリーから選択可能
- Noto Sans JP, Noto Serif JP, Zen Kaku Gothic New など
- 日本語に最適化されたフォントを用意

📑 **目次の自動生成**
- 見出しから自動的に目次を生成
- クリックで該当箇所へスムーズスクロール
- スクロール位置に応じて現在位置をハイライト表示

📁 **ファイルブラウザ**
- フォルダ内のMarkdownファイルを一覧表示
- ファイルの変更を自動検知して再読み込み

⚙️ **その他の機能**
- フォントサイズの調整
- コンテンツ幅の変更
- 設定の自動保存

### 使い方

1. 拡張機能のアイコンをクリック
2. 「フォルダを開く」からMarkdownファイルが含まれるフォルダを選択
3. ファイルリストからプレビューしたいファイルを選択

### プライバシー

この拡張機能は：
- ユーザーデータを外部に送信しません
- インターネット接続を必要としません（Google Fontsの読み込みを除く）
- すべてのデータはローカルに保存されます
