# Chrome Web Store 掲載情報（日本語）

## 拡張機能名
Rich Markdown Preview

## 短い説明（132文字以内）
ローカルのMarkdownファイルを美しくプレビュー。カスタマイズ可能なテーマ、フォント選択、目次の自動生成に対応。

## 詳細な説明

Rich Markdown Previewは、ローカルのMarkdownファイルを美しく読みやすくプレビューするためのChrome拡張機能です。

### 主な機能

📖 **美しいプレビュー**
- Markdownファイルを見やすいフォーマットで表示
- シンタックスハイライト付きのコードブロック
- テーブル、リスト、引用、画像の美しいレンダリング
- 絵文字ショートコード対応（:smile: など）
- フロントマター（YAML）の自動除去

🎨 **20以上のプリセットテーマ**
- GitHub Dark/Light、Dracula、Nord、Tokyo Night、Monokai など人気テーマを多数収録
- 完全にカスタマイズ可能なカラースキーム
- 見出し、リンク、コードブロックなど27種類の要素ごとの色設定
- カスタムテーマの作成・保存・削除

🔤 **フォント・テキスト設定**
- 6種類のフォントファミリーから選択可能
- Noto Sans JP, Noto Serif JP, Zen Kaku Gothic New など
- 日本語に最適化されたフォントを用意
- フォントサイズ、行間、文字間隔の細かな調整
- コンテンツ幅の変更（narrow / medium / wide / full）

📑 **目次の自動生成**
- 見出しから自動的に目次を生成
- クリックで該当箇所へスムーズスクロール
- スクロール位置に応じて現在位置をハイライト表示

📁 **ファイルブラウザ & タブ管理**
- フォルダ内のMarkdownファイルをツリー表示
- ファイルの変更を自動検知して再読み込み
- 複数ファイルのタブ管理（ドラッグ&ドロップで順序変更）
- ファイルの直接ドラッグ&ドロップでプレビュー
- 右クリックコンテキストメニュー（パスコピー、ファイル名コピーなど）

📋 **コピー機能**
- Markdownソースのコピー
- リッチテキスト（HTML）としてのコピー
- ファイルパス・ファイル名のコピー

🔍 **検索機能**
- Ctrl+F / Cmd+F でプレビュー内をテキスト検索
- マッチ数の表示と前後ナビゲーション
- 検索結果のハイライト表示

📝 **拡張Markdown対応**
- Mermaidダイアグラム（フローチャート、シーケンス図など）
- タスクリスト（チェックボックス）
- 脚注（[^1]形式）
- GitHub Flavored Markdown完全対応

⚙️ **その他**
- 設定の自動保存
- キーボードショートカット対応
- 完全オフライン動作（Google Fonts読み込みを除く）

### 使い方

1. 拡張機能のアイコンをクリック
2. 「フォルダを開く」からMarkdownファイルが含まれるフォルダを選択
3. ファイルリストからプレビューしたいファイルを選択

### プライバシー

この拡張機能は：
- ユーザーデータを外部に送信しません
- インターネット接続を必要としません（Google Fontsの読み込みを除く）
- すべてのデータはローカルに保存されます
