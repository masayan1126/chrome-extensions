API Reference

このドキュメントでは、利用可能なすべてのAPIエンドポイントについて説明します。

エンドポイント一覧

メソッドパス説明
GET/api/usersユーザー一覧を取得
POST/api/users新規ユーザーを作成
GET/api/users/:idユーザー詳細を取得
PUT/api/users/:idユーザー情報を更新
DELETE/api/users/:idユーザーを削除

認証

すべてのAPIリクエストには、Bearer Tokenによる認証が必要です。
