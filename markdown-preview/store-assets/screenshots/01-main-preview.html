My Project

高速で安全な次世代Webフレームワーク。TypeScriptで構築され、優れた開発者体験を提供します。

主な機能

  • ⚡ 高速なホットリロード対応の開発サーバー
  • 📦 最適化されたビルド出力
  • 🔧 豊富なプラグインエコシステム
  • 📝 TypeScriptのネイティブサポート

クイックスタート

// プロジェクトの初期化
npx create-my-project my-app
cd my-app
npm run dev
Note: Node.js 18以上が必要です。

詳細はドキュメントをご参照ください。
