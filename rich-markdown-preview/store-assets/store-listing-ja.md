# Chrome Web Store 掲載情報（日本語）

## 拡張機能名
Rich Markdown Preview

## 短い説明（132文字以内）
ローカルMarkdownを最も美しく読む拡張機能。25テーマ、日本語フォント、フォルダ管理、ライブ同期、レビューコメント＆AIエクスポートに対応。

## 詳細な説明

### ローカルMarkdownファイルを、最も美しく、最も自由に読む。

VS CodeやVimで書いたMarkdownを、ブラウザでそのままリッチにプレビュー。
25種類のテーマ、日本語フォント対応、フォルダ丸ごと管理、ファイル変更の自動検知——他のMarkdownプレビュー拡張では手の届かないところまでカバーした、こだわり派のためのChrome拡張機能です。

---

### なぜ Rich Markdown Preview？

多くのMarkdownプレビュー拡張は「1ファイルを開いて表示する」だけ。
Rich Markdown Previewは違います。

- フォルダを開けば、中のファイルがツリーで一覧表示される
- エディタで保存すれば、1秒以内にプレビューが自動で更新される
- 複数ファイルをタブで切り替えながら読める
- 次にChromeを開いたとき、前回の作業状態がそのまま復元される

**「エディタの隣に常に置いておくプレビューア」として設計されています。**

### 主な機能

📖 **美しいMarkdownレンダリング**
- GitHub Flavored Markdown（GFM）完全対応
- 170以上の言語に対応したシンタックスハイライト
- テーブル、リスト、引用、画像の美しいレンダリング
- 絵文字ショートコード対応（:smile: → 😄）
- フロントマター（YAML）の自動除去
- DOMPurifyによるセキュアなHTML出力

🎨 **25種類のプリセットテーマ + カスタムテーマ**
- GitHub Dark/Light、Dracula、Nord、Tokyo Night、Monokai、Catppuccin、Synthwave '84 など人気テーマを網羅
- ライト10種 / ダーク15種、気分や環境に合わせて瞬時に切り替え
- カスタムテーマでは21の色要素を個別設定可能（H1〜H6の見出し色を個別に指定できる拡張はほぼ存在しません）
- アクセントカラー・テキスト色・背景色・ボーダー色の一括設定にも対応
- 作成したカスタムテーマは保存・管理・削除が可能

🔤 **日本語に最適化されたフォント・テキスト設定**
- 日本語フォント5書体をワンクリックで切り替え（Noto Sans JP / Noto Serif JP / Zen Kaku Gothic New / Zen Maru Gothic / M PLUS Rounded 1c）
- フォントサイズ：12〜32pxの範囲で調整
- 行間：1〜3倍の範囲で調整
- 文字間隔：-0.05〜0.2emの範囲で微調整
- コンテンツ幅を4段階（narrow / medium / wide / full）で切り替え
- Google Fontsからの動的読み込みで、初回以降はキャッシュで高速表示

📁 **フォルダ管理 & エディタ連動**
- フォルダを選択するだけでMarkdownファイルをツリー表示（最大5階層）
- .md / .markdown / .mdx / .mkd に対応
- 外部エディタでの保存を1秒以内に自動検知してプレビュー更新
- 隠しファイル（.で始まるファイル）の表示/非表示切り替え
- ディレクトリのリフレッシュボタンで最新状態を即時反映
- 右クリックでファイルパス・ファイル名をコピー

📑 **タブ管理 & セッション復元**
- 複数ファイルをブラウザライクなタブで管理
- タブのドラッグ&ドロップで自由に並べ替え
- ファイルをブラウザにドラッグ&ドロップで即座にタブで開く
- 前回開いていたフォルダ・タブを次回起動時に自動復元（IndexedDBで永続化）

📑 **目次の自動生成**
- H1〜H6の見出しから目次を自動構築
- クリックで該当箇所へスムーズスクロール
- スクロール位置に応じて現在の見出しをリアルタイムでハイライト
- 目次パネルも現在位置に自動スクロール
- 表示/非表示をワンクリックで切り替え

🔍 **文書内検索**
- Ctrl+F / Cmd+F で検索パネルを起動
- マッチ数と現在位置を「X / Y」形式で表示
- 前後のマッチへのナビゲーション（Enter / Shift+Enter）
- マッチ箇所のハイライト表示 & アクティブマッチの強調
- 検索結果への自動スクロール

📝 **拡張Markdown対応**
- Mermaidダイアグラム（フローチャート、シーケンス図、ガントチャートなど）
- ダーク/ライトテーマに応じてMermaidも自動でテーマ切替
- タスクリスト（チェックボックス表示）
- 脚注（[^1] 形式で脚注セクション自動生成）
- details / summary による折りたたみ要素

💬 **レビューコメント & AIエクスポート**
- プレビュー上でテキストを選択してレビューコメントを追加
- 「修正」「削除」の2種類のコメントタイプに対応
- コメントの未解決/解決済みステータス管理
- コメント対象箇所のハイライト表示 & クリックで該当箇所へスクロール
- レビューコメントをAIエージェント向け指示書としてエクスポート（Markdown / JSON形式）
- ファイルごと・セクションごとに整理された構造化出力
- クリップボードコピー & ファイルダウンロードに対応
- IndexedDBでコメントを永続化、次回起動時に自動復元

📋 **コピー機能**
- Markdownソースをワンクリックでコピー
- 変換済みHTMLとしてコピー（他のアプリに貼り付けるときに便利）
- ファイルパス・ファイル名のコピー

⌨️ **キーボードショートカット**
- Ctrl+F / Cmd+F：検索を開く
- Escape：検索を閉じる
- Enter：次のマッチへ
- Shift+Enter：前のマッチへ

⚙️ **設定の自動保存**
- テーマ、フォント、サイズ、行間、字間、コンテンツ幅、目次の表示状態などすべての設定を自動保存
- Chrome Storage APIで永続化、次回起動時に即座に復元

### 他のMarkdownプレビュー拡張との違い

| | Rich Markdown Preview | 一般的な拡張 |
|---|---|---|
| ディレクトリツリー表示 | ✅ フォルダ丸ごと管理 | ❌ 単一ファイルのみ |
| ファイル変更の自動同期 | ✅ 1秒以内に反映 | ❌ 手動リロード |
| タブ管理 | ✅ 複数ファイルを同時に開ける | ❌ 1ファイルずつ |
| セッション復元 | ✅ フォルダ・タブを自動復元 | ❌ 毎回開き直す |
| テーマ | ✅ 25種類 + カスタム無制限 | △ 数種類 |
| 色のカスタマイズ | ✅ 21要素を個別設定 | △ 背景色程度 |
| 日本語フォント | ✅ 5書体対応 | ❌ 未対応が多い |
| Mermaidダイアグラム | ✅ テーマ自動切替 | ❌ 未対応が多い |
| 目次 + スクロール同期 | ✅ リアルタイム追従 | △ 静的な目次 |
| レビューコメント | ✅ テキスト選択で即追加 + AI指示書エクスポート | ❌ 未対応 |

### 使い方

1. 拡張機能のアイコンをクリック
2. 「フォルダを開く」からMarkdownファイルが含まれるフォルダを選択
3. ファイルリストからプレビューしたいファイルを選択
4. お好みのテーマやフォントを設定してお楽しみください

💡 ファイルを直接ドラッグ&ドロップで開くこともできます。

---

### プライバシー

この拡張機能は：
- ユーザーデータを外部に送信しません
- インターネット接続を必要としません（Google Fontsの読み込みを除く）
- すべてのデータはローカルに保存されます
- DOMPurifyによりMarkdown内の悪意あるスクリプトを無害化します
