# Privacy Policy for Rich Markdown Preview

**Last Updated: 2024**

## Overview

Rich Markdown Preview is a Chrome extension that allows users to preview local Markdown files. This privacy policy explains how the extension handles user data.

## Data Collection

### What We Collect

**We do not collect any personal data.**

The extension only stores the following information locally on your device:
- Theme preferences (selected theme, custom themes)
- Display settings (font size, font family, content width)
- No personal information is ever collected

### What We Don't Collect

- Personal information (name, email, etc.)
- Browsing history
- File contents
- Usage analytics
- Location data

## Data Storage

All data is stored locally using:
- Chrome's `storage.local` API for extension settings
- Browser's localStorage for web app mode

**No data is transmitted to external servers.**

## Third-Party Services

### Google Fonts

The extension may load fonts from Google Fonts when a non-system font is selected. This is the only external connection made by the extension. Please refer to [Google's Privacy Policy](https://policies.google.com/privacy) for information about how Google handles data.

## Permissions

The extension requires the following permissions:
- `storage`: To save user preferences locally

## Data Sharing

We do not share, sell, or transfer any user data to third parties.

## Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated revision date.

## Contact

If you have any questions about this privacy policy, please create an issue on our GitHub repository.

# プライバシーポリシー（日本語）

**最終更新日: 2024年**

## 概要

Rich Markdown Previewは、ローカルのMarkdownファイルをプレビューするためのChrome拡張機能です。このプライバシーポリシーでは、本拡張機能がユーザーデータをどのように扱うかについて説明します。

## データ収集

### 収集するデータ

**個人データは一切収集しません。**

本拡張機能は、以下の情報のみをユーザーのデバイス上にローカル保存します：
- テーマ設定（選択したテーマ、カスタムテーマ）
- 表示設定（フォントサイズ、フォントファミリー、コンテンツ幅）
- 個人情報は一切収集されません

### 収集しないデータ

- 個人情報（名前、メールアドレスなど）
- 閲覧履歴
- ファイルの内容
- 使用状況の分析
- 位置情報

## データ保存

すべてのデータは以下の方法でローカルに保存されます：
- 拡張機能の設定にはChromeの `storage.local` API
- WebアプリモードではブラウザのlocalStorage

**外部サーバーにデータが送信されることはありません。**

## サードパーティサービス

### Google Fonts

システムフォント以外のフォントが選択された場合、Google Fontsからフォントを読み込むことがあります。これが本拡張機能が行う唯一の外部接続です。Googleがデータをどのように扱うかについては、[Googleのプライバシーポリシー](https://policies.google.com/privacy)をご参照ください。

## 権限

本拡張機能は以下の権限を必要とします：
- `storage`: ユーザー設定をローカルに保存するため

## データ共有

ユーザーデータを第三者と共有、販売、または転送することはありません。

## ポリシーの変更

このプライバシーポリシーは随時更新される場合があります。変更があった場合は、更新日を記載してこのページに掲載します。

## お問い合わせ

このプライバシーポリシーに関するご質問は、GitHubリポジトリでissueを作成してください。
