# Privacy Policy - Chat Stamp Preview

Last updated: 2026-02-01

## Overview

Chat Stamp Preview is a Chrome extension that provides a chat-style stamp preview tool. This privacy policy explains how the extension handles user data.

## Data Collection

**Chat Stamp Preview does not collect, transmit, or share any user data.**

- No personal information is collected
- No usage analytics or tracking
- No data is sent to external servers
- No cookies are used
- No network requests are made

## Data Storage

Uploaded stamp images are stored locally in your browser using IndexedDB. This data:

- Remains entirely on your device
- Is never transmitted to any server
- Can be deleted at any time using the "Delete All Stamps" button within the extension
- Is automatically removed when the extension is uninstalled

## Permissions

This extension requires **no special browser permissions**. It operates entirely within its own extension page.

## Third-Party Services

This extension does not use any third-party services, libraries, or APIs.

## Changes to This Policy

Any changes to this privacy policy will be reflected in the extension update notes on the Chrome Web Store.

## Contact

If you have questions about this privacy policy, please contact us through the Chrome Web Store support page.

# プライバシーポリシー - Chat Stamp Preview

最終更新日: 2026-02-01

## 概要

Chat Stamp Previewは、チャット風のスタンププレビューツールを提供するChrome拡張機能です。本プライバシーポリシーは、本拡張機能におけるユーザーデータの取り扱いについて説明します。

## データ収集

**Chat Stamp Previewは、ユーザーデータの収集、送信、共有を一切行いません。**

- 個人情報の収集なし
- 利用統計やトラッキングなし
- 外部サーバーへのデータ送信なし
- Cookieの使用なし
- ネットワークリクエストなし

## データ保存

アップロードされたスタンプ画像は、ブラウザのIndexedDBを使用してローカルに保存されます。このデータは：

- 完全にお使いのデバイス上に留まります
- サーバーに送信されることはありません
- 拡張機能内の「全スタンプ削除」ボタンでいつでも削除できます
- 拡張機能のアンインストール時に自動的に削除されます

## 権限

本拡張機能は**特別なブラウザ権限を必要としません**。拡張機能自身のページ内でのみ動作します。

## サードパーティサービス

本拡張機能は、サードパーティのサービス、ライブラリ、APIを使用していません。

## ポリシーの変更

本プライバシーポリシーの変更は、Chrome Web Storeの拡張機能更新ノートに反映されます。

## お問い合わせ

本プライバシーポリシーに関するご質問は、Chrome Web Storeのサポートページよりお問い合わせください。
