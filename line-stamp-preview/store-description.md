## 拡張機能名
Chat Stamp Preview

## 短い説明 (132文字以内)

### 日本語
チャット風UIでスタンプ画像をプレビューできるツール。PNG画像をアップロードし、送信者・受信者の会話をシミュレーションできます。

### English
Preview your stamp images in a chat-style UI. Upload PNG images and simulate conversations between sender and receiver.

## 詳細な説明

### 日本語
Chat Stamp Previewは、チャット風のインターフェースでスタンプ画像をプレビューできるツールです。

主な機能:
- PNG画像をドラッグ＆ドロップまたはファイル選択でアップロード
- 送信者・受信者を切り替えて会話をシミュレーション
- チャット画面を画像（PNG）として書き出し
- アップロードしたスタンプはブラウザに自動保存（再利用可能）
- キーボードショートカット対応（Ctrl+Z で取り消し、Tab で送信者切り替え）

スタンプや絵文字を制作する方のプレビュー確認に最適です。

特徴:
- 完全オフライン動作（外部サーバーへのデータ送信なし）
- 特別な権限不要
- 軽量・高速

### English
Chat Stamp Preview is a tool for previewing stamp images in a chat-style interface.

Key Features:
- Upload PNG images via drag & drop or file picker
- Switch between sender and receiver to simulate conversations
- Export the chat screen as a PNG image
- Uploaded stamps are saved in the browser for reuse
- Keyboard shortcuts supported (Ctrl+Z to undo, Tab to switch sender)

Perfect for stamp and emoji creators to preview their designs in a realistic chat context.

Highlights:
- Fully offline (no data sent to external servers)
- No special permissions required
- Lightweight and fast
