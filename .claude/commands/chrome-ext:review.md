# Chrome Extension Review Response

Chrome Web Storeのレビューへの返信を作成します。

## 実行内容

以下のファイルを参照して、レビューへの対応を行ってください：

1. **ワークフロー**: `.claude/skills/chrome-ext-review/WORKFLOW.md`
2. **返信テンプレート**: `.claude/skills/chrome-ext-review/TEMPLATES.md`

## プロセス

### Step 1: 情報収集
ユーザーに以下を確認：
- 対象の拡張機能名
- レビューの内容（コピー＆ペースト）
- レビューの評価（星の数）
- レビューの言語

### Step 2: レビュー分類
- ポジティブ（5-4つ星）
- ネガティブ（3-1つ星）
- 問題のカテゴリ（バグ/機能要望/UX/誤解など）

### Step 3: 返信作成
テンプレートを参考に、以下を含む返信を作成：
- 感謝の表明
- 問題への対応（該当する場合）
- 次のアクション
- サポート案内

### Step 4: 改善提案（ネガティブレビューの場合）
- 根本原因の分析
- 改善提案の作成
- 優先度の判定

日本語と英語の両方の返信案を提示してください。
