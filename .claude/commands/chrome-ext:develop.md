# Chrome Extension Development

Chrome拡張機能の新規開発を行います。

## 実行内容

以下のファイルを参照して、ステップバイステップで開発を進めてください：

1. **ワークフロー**: `.claude/skills/chrome-ext-develop/WORKFLOW.md`
2. **テンプレート**: `.claude/skills/chrome-ext-develop/TEMPLATES.md`
3. **技術リファレンス**: `.claude/skills/chrome-ext-develop/REFERENCE.md`

## プロセス

### Phase 1: 要件収集
ユーザーに以下を確認：
- 拡張機能のコンセプト
- 主要な機能要件
- ターゲットユーザー

### Phase 2: 競合調査
- Chrome Web Storeで類似拡張機能を調査
- 差別化ポイントを特定
- 競合分析レポートを作成

### Phase 3: 技術設計
- ディレクトリ構造の設計
- manifest.json の設定
- 必要な権限の特定

### Phase 4: 開発
- コア機能の実装
- 多言語対応（ja/en）
- テスト

### Phase 5: ストア申請準備
以下をユーザーと一緒に準備：
- [ ] アイコン画像の仕様説明（16, 48, 128px PNG）
- [ ] スクリーンショット仕様（1280x800）
- [ ] プロモーション画像仕様（440x280, 1400x560）
- [ ] store-listing-ja.md 作成
- [ ] store-listing-en.md 作成
- [ ] PRIVACY_POLICY.md 作成
- [ ] ZIPファイルのビルド

各ステップでユーザーに確認を取りながら進めてください。
